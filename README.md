# 🌟 Focus：基于生理信号的智能学习助手

> **让每一次学习，都更有方向、更少焦虑。**

Focus 是一款融合 **生理信号分析** 与 **智能学习建议** 的创新学习助手。通过实时监测心率变异性（HRV），科学评估学习者的专注度与压力水平，提供个性化反馈，帮助用户提升学习效率、管理情绪状态。

---

## 📌 核心功能

- ✅ **番茄学习计时**：标准 25 分钟专注学习周期  
- 📈 **Stress 实时检测**：基于 HRV 计算压力值，识别焦虑状态  
- 🎯 **Focus 评分系统**：量化专注度得分（0–100）  
- 📊 **学习报告生成**：自动汇总学习时长、专注度、压力趋势  
- 💡 **智能学习建议**：动态推荐休息、深呼吸或调整节奏  
- 🔒 **本地数据存储**：安全记录历史会话，支持长期追踪

> 系统实现 **“感知 → 分析 → 反馈”** 的完整闭环。

---

## 🧪 演示流程

1. 打开应用，进入主界面  
2. 开始一次番茄学习（25 分钟）  
3. 实时观察 Stress 压力曲线变化  
4. 学习结束，自动停止计时  
5. 查看生成的 **Study Report**（含专注度、压力、时长）  
6. 接收个性化 **学习建议**（如：“建议休息 5 分钟”）
